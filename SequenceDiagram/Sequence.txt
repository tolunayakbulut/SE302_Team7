When the user runs the program, he or she will see the following options, open an existing tree, create a tree and join trees. If user selects the open tree option user will see previously created tree. If there is no previously created tree, the user will select create a tree and inside this tree user will have to add at least two family members and fill their mandatory information. After creating the tree user will be able to add more people, delete people and edit the information of the people who are inside of this tree. If the user choses join then he or she will choose 2 people to couple, then the families' of these people will be joined.
